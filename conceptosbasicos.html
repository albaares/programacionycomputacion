Conceptos clave de la programación

Variables, condicionales, bucles y funciones

Los 4 pilares de la programación

Para construir cualquier programa, desde el más simple al más complejo, necesitamos dominar cuatro conceptos fundamentales.

Variables
Condicionales
Bucles
Funciones

1. Las cajas de la información: Variables

  • Son contenedores donde guardamos datos para usarlos más tarde.
  • Cada variable tiene un nombre único para poder identificarla.
  • Pensemos en ellas como cajas etiquetadas donde guardamos un valor.

Ejemplo de variables

Podemos guardar diferentes tipos de datos:

// Guardamos texto (string)
nombre = "Ana"
// Guardamos un número entero
edad = 17
// Guardamos un número decimal
altura = 1.65
// Guardamos un valor de verdad
es_estudiante = verdadero

2. Tomando decisiones: Condicionales

  • Permiten que nuestro programa elija un camino u otro según se cumpla una condición.
  • Son la base de la lógica en la programación.
  • La estructura más común es si (if) ... si no (else).
"Si está lloviendo, entonces coge el paraguas. Si no, coge las gafas de sol."

Ejemplo de condicional

Comprobamos si una persona es mayor de edad:

edad = 17

si (edad >= 18) entonces
  mostrar "Puedes pasar"
si no
  mostrar "No puedes pasar, eres menor"
fin si

// Resultado: "No puedes pasar, eres menor"

3. Repitiendo tareas: Bucles

  • Nos permiten ejecutar un bloque de código varias veces sin tener que reescribirlo.
  • Son increíblemente útiles para procesar listas de datos o realizar acciones repetitivas.
  • Los más comunes son el bucle para (for) y mientras (while).

Ejemplo de bucle

Contamos del 1 al 5 usando un bucle "para":

para i desde 1 hasta 5
  mostrar "El número es: " + i
fin para

// Resultado:
// El número es: 1
// El número es: 2
// El número es: 3
// El número es: 4
// El número es: 5

4. Recetas reutilizables: Funciones

  • Son bloques de código con un nombre que realizan una tarea específica.
  • Las creamos una vez y podemos "llamarlas" o usarlas tantas veces como queramos.
  • Ayudan a organizar y simplificar nuestro programa.

Ejemplo de función

Creamos una función para saludar y la usamos varias veces:

// 1. Definimos la "receta"
función saludar(nombre)
  mostrar "¡Hola, " + nombre + "! ¿Cómo estás?"
fin función

// 2. La usamos cuando la necesitamos
saludar("Ana")
saludar("Carlos")

// Resultado:
// ¡Hola, Ana! ¿Cómo estás?
// ¡Hola, Carlos! ¿Cómo estás?

Juntando las piezas

  1. Usamos variables para guardar los datos.
  2. Con condicionales, tomamos decisiones basadas en esos datos.
  3. Con bucles, repetimos acciones sobre los datos.
  4. Y con funciones, organizamos todo en bloques lógicos y reutilizables.
